This is a personal project built from my interest in space and the solar system. It was intended to be a interactive web-based program to absorb a basic understanding of the solar system while still providing an entertaining objective. Using object-oriented programming, I added the planets and stars into the universe, giving them unique mass and velocities for orbital trajectories. This was displayed using the p5.js library.
After seeing the web-interface and a lack of mobile integration for the printing application on campus, I and a team of User-Interface design classmates decided to create a mobile interface and redesign the web-application. This project used HTML, JavaScript and CSS for the web-interface, and an online tool called Figma to design the mobile interface. This was meant to be exclusively a design project, so there is not functionality on these interfaces yet.
The 'AIM' in Project AIM stands for "Artificial Intelligence Manager". This project is an attempt to make an automatic podcast manager, which switches cameras based on who is talking. This was a team project, in a class that was meant to simulate a working environment with customers, employees and executives. The AI part of this project currently is a facial landmark detection system in which finds the positions of a podcaster's mouth, and detects if they are speaking or not based on the change of the lip positions. This, in combination with a algorithmic audio level reflex-agent makes decisions based on two 'confidence variables' that are calculated by the systems described earlier. This is a work in progress.
